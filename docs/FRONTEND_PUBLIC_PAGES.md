# Frontend Public Pages

## 首页

默认首页保留管理员自定义 `home_content` 的优先级；未配置自定义内容时，展示参考 Google Antigravity 风格的浅色产品首页。页面包含：

- 白底极简顶部导航、站点品牌、中部导航、右上角语言切换和站内控制台入口。
- 首屏超大居中标题，以及从中心向外升空散开的工具图标云动画（`.ag-hero-orbit`、`.ag-tool`、`.ag-liftoff-core`）。
- 默认首页文案跟随语言切换；中文环境下导航、主标题、预览窗口、工具图标、能力卡片和主题按钮均显示中文文案，顶部和底部语言切换均可切换中英文。
- 默认首页支持浅色/深色主题切换，底部主题按钮会同步切换页面根节点配色、标题文字、预览窗口和能力卡片。
- 站内开始使用/控制台入口和 `/models` 模型列表入口；首页不提供下载功能语义。
- 浅色产品预览窗口和产品能力卡片，沿用原首页文案体系说明“一键接入”和“用多少付多少”等能力；不再展示额外副文案说明、账号调度说明和模型目录说明。

## 模型列表页

`/models` 是公开展示页，数据来自前端 `useModelWhitelist.ts` 中维护的模型集合。页面提供：

- 平台分组筛选。
- 模型名称搜索。
- 模型类型标签，用于快速区分 chat、fast、reasoning、coding、multimodal。

注意：该页面只展示前端内置模型集合，不代表某个用户或账号当前一定可用；实际可用性仍由渠道、账号池、模型映射和后端配置决定。
